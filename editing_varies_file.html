La munición es necesaria para un arma que tenga la propiedad Munición. La descripción de un arma especifica el tipo de munición que utiliza el arma. La tabla de Munición lista los diferentes tipos y la cantidad que obtienes al comprarlos. La tabla también enumera el objeto que se utiliza normalmente para almacenar cada tipo; el Contenedor debe comprarse por separado.
